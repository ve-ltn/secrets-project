# Secrets

A web application that fetches and displays random secrets from a public API — built with Node.js, Express, and EJS.

## Live Demo

🔗 [secrets-project-production-8409.up.railway.app](https://secrets-project-production-8409.up.railway.app)

## Features

- Fetches a random secret from a public API on every page load
- Displays the secret along with the username of who submitted it
- Clean and responsive UI

## Tech Stack

- Node.js
- Express.js
- EJS
- Axios
- Railway (deployment)

## Try It Out!

You can try the application live at the demo link above. Simply open the page and a new random secret will be displayed each time you visit or refresh!
